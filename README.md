## 🛠 Technology Stack
- **React 19** — Frontend library
- **Vite** — Fast build tool
- **Tailwind CSS** — Utility-first CSS framework
- **Redux Toolkit** — State management
- **React Router DOM** — Routing
- **Framer Motion** — Animations
- **Lucide React** — Icons
- **TypeScript** — Static typing
